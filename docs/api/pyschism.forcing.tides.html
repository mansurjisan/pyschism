class pyschism.forcing.tides.tides.Tides(elevation: bool = True, velocity: bool = False, database: Union[str, pyschism.forcing.tides.tides.TidalDatabase] = <TidalDatabase.TPXO: <class 'pyschism.forcing.tides.tpxo.TPXO'>>)¶

Bases: pyschism.forcing.tides.bctypes.BoundaryCondition

Main class for requesting tidal boundary forcing for a SCHISM run.

The user should add tidal requests to this class via it’s methods.

Parameters

  • elevation (optional) – Apply elevation forcing to boundary, defaults to True.

  • velocity (optional) – Apply velocity forcing to boundary (currently disabled), defaults to False.

  • database (optional) – Tidal database to use in order to obtain boundary initial conditions, defaults to TidalDatabase.TPXO
